Position Properties

Box 1 Static 
    position: static;
    top: 100px;
    /* top will not work */
Box 2 Relative 
    position: relative;
    top: 150px;
    left: 500px;
Box 3 Fixed 
    position: fixed;
    /* z index become 1,
    and remain fixed  */
Box 4 Absolute 
    position: absolute;
    left: 100px;
    top: 200px;
    /* relative to nearest
    positioned ancestor or
    body */
Box 5 Sticky 
    position: sticky;
    top: 300px;
    left: 400px;
    /* toggle between
    relative and fixed */
    /* relative at normal
    state, fixed after
    scroll */
